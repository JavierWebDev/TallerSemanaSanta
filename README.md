# Taller Semana Santa

Trabajo práctico correspondiente a la materia **Lógica de Programación**. Este repositorio contiene una serie de ejercicios desarrollados en Java como parte de las actividades del taller.

## Tecnología

- **Lenguaje:** Java

## Estructura

El taller está compuesto por 20 ejercicios (`ejercicio1.java` – `ejercicio20.java`), cada uno abordando distintos conceptos y problemas propuestos en el taller.
